# Variable: ANALYTICS\_UNAUTHENTICATED\_USER

> `const` **ANALYTICS\_UNAUTHENTICATED\_USER**: `"UNAUTHENTICATED"` = `"UNAUTHENTICATED"`

Explicit sentinel for analytics events that intentionally have no
authenticated user (e.g. pre-auth landing pages).
Pass as `user_ulid` to bypass the "missing user" guard.
