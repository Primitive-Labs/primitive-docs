# PrimitiveStaticLayout

Static “carded” page layout for simple pages outside the main app shell.

Provides a header with app branding and a centered card container for the route content.

---

## Slots

| Name    | Description                                                                         | Bindings |
| ------- | ----------------------------------------------------------------------------------- | -------- |
| default | Optional override for the page content area. Defaults to rendering a `router-view`. |          |
