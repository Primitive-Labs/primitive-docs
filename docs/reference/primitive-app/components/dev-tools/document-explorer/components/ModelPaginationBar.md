# ModelPaginationBar

Model Pagination Bar - Bottom bar with page navigation, page size selector,
and record count display for the records table in DocumentExplorerModel.

---

## Props

| Prop name   | Description | Type   | Values | Default |
| ----------- | ----------- | ------ | ------ | ------- |
| currentPage |             | number | -      |         |
| totalCount  |             | number | -      |         |
| pageSize    |             | number | -      |         |
| totalPages  |             | number | -      |         |
