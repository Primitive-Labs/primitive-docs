# DocumentSharingSection

Document Sharing Section - Manages sharing permissions and invitations
for a document in the Document Properties Panel.

Shows existing permissions with avatar initials, pending invitations
with cancel option, and an invite collaborator form.

---

## Props

| Prop name  | Description | Type    | Values | Default |
| ---------- | ----------- | ------- | ------ | ------- |
| documentId |             | string  | -      |         |
| canWrite   |             | boolean | -      |         |
