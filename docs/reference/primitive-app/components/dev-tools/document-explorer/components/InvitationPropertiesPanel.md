# InvitationPropertiesPanel

Invitation Properties Panel - Right collapsible panel shown when a pending
invitation is selected in the Document Explorer left sidebar.

Shows invitation title with "Pending" badge, document ID, accept/decline
buttons, and invitation metadata (invited by, permission, date).

---

## Props

| Prop name  | Description | Type              | Values | Default |
| ---------- | ----------- | ----------------- | ------ | ------- |
| invitation |             | PendingInvitation | -      |         |
