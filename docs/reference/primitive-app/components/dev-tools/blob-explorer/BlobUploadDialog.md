# BlobUploadDialog

Blob Upload Dialog — modal for uploading a single file to a document's blob store.

Features: drag-and-drop drop zone, file info panel (name, size, SHA-256),
confirm/cancel buttons. Upload progress is shown at the bottom of the screen
after the dialog closes.
