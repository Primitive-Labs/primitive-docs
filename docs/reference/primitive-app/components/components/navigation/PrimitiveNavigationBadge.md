# PrimitiveNavigationBadge

Small pill-style badge for navigation items (e.g. counts, labels).

---

## Props

| Prop name | Description                                                                             | Type        | Values | Default |
| --------- | --------------------------------------------------------------------------------------- | ----------- | ------ | ------- |
| badge     | Badge configuration (content/count + color variant). When omitted, nothing is rendered. | [BadgeConfig](/reference/primitive-app/types/symbols/BadgeConfig) | -      |         |
