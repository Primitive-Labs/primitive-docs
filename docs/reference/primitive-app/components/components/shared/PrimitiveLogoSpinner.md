# PrimitiveLogoSpinner

Full-screen loading overlay that shows a spinning Primitive logo.

Commonly used as the fallback UI while auth flows or boot-time async work runs.

---

## Props

| Prop name | Description                                                            | Type   | Values | Default |
| --------- | ---------------------------------------------------------------------- | ------ | ------ | ------- |
| className | Optional extra class(es) applied to the full-screen overlay container. | string | -      |         |
| size      | Size of the logo image in pixels.<br/>`@default` 128                   | number | -      | 128     |
