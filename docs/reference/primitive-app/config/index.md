# Config

Primitive App is configured primarily via the `createPrimitiveApp()` bootstrap options.

This section documents the three most common configuration “surfaces”:

- **Navigation config**: the sidebar + bottom nav structure (`NavigationConfig`)
- **App config**: app identity and shell behavior (`InitializeAppConfigOptions`)
- **Env config**: environment variables Primitive App optionally reads (`ImportMetaEnv`)
